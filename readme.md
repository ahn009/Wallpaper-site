# Luxe Wallpapers & Vinyl

**Professional Wallpaper & Vinyl Installation in Richmond, TX**

A premium, minimal, and fully responsive static website for Luxe Wallpapers & Vinyl – delivering flawless, high-end wallpaper and vinyl installations.

Live Demo: [https://your-domain.com](https://your-domain.com) *(replace with your actual URL when deployed)*

## Features

- Elegant, premium, and minimal design with luxurious gold accents
- Fully responsive across all devices (desktop, tablet, mobile)
- Prominent Call-to-Action buttons ("Call Now" and "Free Quote / Consultation") immediately visible on homepage
- Dynamic logo scaling that adapts beautifully to screen size
- Font Awesome 6 icons for clean, modern visual enhancement
- Responsive navigation with uniform, adaptive spacing between links
- Integrated Google Maps in footer for easy location access
- Google Tag Manager included for analytics
- SEO-optimized with proper meta tags and structured data
- Fast-loading static site with local assets

## Tech Stack

- HTML5
- CSS3 (`style.css` + `responsive.css`)
- Vanilla JavaScript + jQuery + Bootstrap
- Font Awesome 6 (for all icons including social media)
- No heavy frameworks – pure static site for maximum speed and ease of hosting
